# 场景 F：NGA 龙头战法 — 情绪周期判断

**触发语**：「现在市场什么阶段」「龙头战法说现在该重仓还是轻仓」「最近市场怎么判断」「情绪周期是什么」

---

## 情绪周期四阶段框架

| 阶段 | 判断信号 | 仓位建议 |
|------|---------|---------|
| **迷茫期** | 前龙见顶，新龙未起，亏钱效应蔓延 | 空仓观望 |
| **上升期** | 龙头确立，板块联动，连板高度新高 | 重仓主升 70-90% |
| **补涨期** | 高位股大涨，后排轮动，人气高峰 | 降仓 30-50%，去弱换强 |
| **退潮期** | 后排崩塌，亏钱效应扩散，高位股分歧 | 清仓离场 < 10% |

---

## 判断步骤

**第一步：获取数据**

```bash
python3 scripts/scan_dragon.py --mode emotion
```

**第二步：读取三个核心指标**

```
涨停家数     → 情绪温度计
  < 20 家：冰点/迷茫  20-50 家：偏暖  50-100 家：正常  > 100 家：高潮

连板最高高度  → 市场高度（最强股连板数）
  高度持续新高 → 上升期信号
  高度见顶回落 → 退潮先行信号

高位股表现   → 主升/分歧/崩板
  强势封板（顶板）       → 延续上升
  巨量分歧（开板换手）    → 高潮见顶信号
  炸板/跌停出货          → 退潮确认
```

**第三步：综合判断**

```
情绪判断输出格式：

当前阶段：[迷茫期 / 上升期 / 补涨期 / 退潮期]（参考）

依据：
  涨停家数：XX 家 → [偏冷/偏暖/高潮]
  市场高度：XX板 [连续新高 / 持平 / 回落]
  高位股状态：[强势顶板 / 分歧放量 / 崩板破位]

仓位建议：XX%
  [迷茫期] → 空仓，等待新龙确立
  [上升初期] → 30-50% 试仓，跟踪龙头
  [上升主升] → 70-90% 重仓主线
  [补涨期] → 降至 30%，持龙头卖跟风
  [退潮期] → 清仓或 < 10% 防守
```

## 详细规则参考

完整情绪周期理论见 → `references/nga-dragon-method.md`，第四章：情绪周期与仓位管理。

关键规则摘录：
- 做龙头不做跟风：只持仓第一个涨停（首板）的强势品种
- 情绪高潮绝不追高：跌停家数 > 30 家 + 大量股票涨停后次日低开 → 坚决离场
- 冰点不悲观：冰点是布局机会，等前期强势板块缩量止跌再进
