# NGA 龙头战法完整参考手册

> 本战法源自 NGA 论坛游资「交易猿」多年实盘总结，是国内短线情绪流的核心操盘体系。
> 核心逻辑：**做最强板块的最强个股**，跟随市场赚钱效应，不拘形式，不拘风格。

---

## 一、核心理念

### 1.1 龙头思想

**龙头 = 一段赚钱效应的载体 + 市场中赚钱效应最突出的个股**

龙头的两大核心属性：
- **启动早**：先于指数上涨，先于板块上涨
- **领涨**：不仅自己涨，还带动相关板块个股同涨

当龙头走出来后，市场形成共识，龙头股成为赚钱效应标杆。资金围绕龙头股的各种属性（题材/板块/关联股）反复挖掘补涨。

---

### 1.2 顺势三原则

**① 顺情绪周期（最重要）**

完整情绪周期的四个阶段：

| 阶段 | 特征 | 操作 |
|------|------|------|
| **迷茫期** | 前龙见顶、新龙酝酿 | 轻仓观望，等待方向 |
| **上升期** | 龙头确立、板块联动 | 重仓参与主升 |
| **补涨期** | 龙头筑顶、后排补涨 | 去弱换强，降仓位 |
| **退潮期** | 后排崩塌、龙头见顶 | 空仓离场 |

**② 顺指数/板块趋势**

共振出妖股：**指数 × 板块 × 情绪三者共振 = 牛股诞生**

- 指数大底通常有主线带领反弹：大资金参与大市值个股带动板块，游资打板块内小票高度
- 高度股/龙头股依附于板块而诞生
- 板块强度判断：**抗跌 + 领涨 + 高度股 + 连板家数**

**③ 顺风格（破执）**

打破三个执念：
- 不拘泥于主升形态（只要涨幅前排、趋势不破就持有）
- 不拘泥于分时形态（龙头分时没有标准答案）
- 不拘泥于 K 线形态（技术形态只是辅助，核心是抗跌 + 领涨）

## 二、择时体系

### 2.1 情绪择时（核心指标：涨跌家数，通达信代码 880005）

| 情绪状态 | 判断标准 | 操作 |
|---------|---------|------|
| **高潮日** | 上涨家数 > 3500 家（普涨） | 去弱换强，高抛跟风股，调仓进龙头 |
| **正常** | 上涨家数 1000-3500 家 | 按日内盘面机动 |
| **冰点日** | 上涨家数 < 1000 家 | 低吸强势股，博弈情绪反包 |
| **极端冰点** | 上涨家数 < 500 家 或连续两天冰点 | 确定性反转信号，重点布局 |

**冰点日的两个买点**：
1. **冰点日当天**：低吸市场强势股（抗跌/逆势走强的）
2. **冰点日次日**：指数二次探底时，买与指数共振反弹的转势票

**情绪信号组合规律**：
```
a. 连续下跌 + 冰点 = 反弹（涨则加仓，跌则减仓）
b. 盘整 + 恐慌 = 大跌风险信号
c. 上涨 + 恐慌 = 继续上涨机会
```

---

### 2.2 板块择时（爆发日 → 分歧日 → 回流日 三段论）

| 关键节点 | 特征 | 操作 |
|---------|------|------|
| **爆发日** | 板块集体大涨，后排股大涨 | 先上车，重仓大票或分仓打板前排 |
| **分歧日**（爆发次日） | 后排股冲高回落，前排股继续走强 | 去弱换强：卖后排，买前排龙头 |
| **回流日** | 分歧后资金二次入场（"倒车接人"） | 加仓领涨股，爆发日成交超100亿则大概率回流 |

判断回流强度：爆发日板块内涨停个股总成交金额 > 100 亿 → 大概率有回流日

---

### 2.3 指数择时（底部寻龙）

- 指数大跌期间，寻找**逆势抗跌的板块和个股**
- 回调级别越大 → 做多力量压制越极致 → 转势后行情越大
- 底部区域策略：逆势抗跌板块 → 区间统计筛强势股 → 爆发日上车

---

### 2.4 前龙见顶日（换龙节点）

妖股起止无缝对接：**前龙头见顶 → 以该日为起点统计涨幅排名 → 很快找到新周期领涨股**

操作：前龙见顶后，立即开始监控各板块涨幅排名，5日内通常能确认新龙头。

## 三、选股方法

### 3.1 核心筛选标准

**① 提前启动 + 领涨**（最核心）
- 大盘下跌时不跌或小跌
- 大盘止跌时企稳或小涨  
- 大盘上涨时开始大涨

**② 极限成交量（量价核心）**

横向对比：
- 按日成交金额排行，关注**近期突然连续出现在两市前列**的个股（排除常年大蓝筹）
- 同一板块多只个股同时出现在成交前列 → 该板块是明星板块，更值得关注

纵向对比：
- 个股自身对比：近期成交出现**历史罕见爆量** → 大行情前兆
- 注意：历史最高量是贯穿整个主升浪的**持续堆量**，不是单天放量

**③ 加分项（内因）**

| 内因 | 说明 |
|------|------|
| 市值小 | 盘子轻，容易被资金推动 |
| 题材正 | 契合当前市场热点/政策方向 |
| 叠加题材多 | 多重题材共振，想象空间大 |
| 股价低 | 绝对价格低，跟进成本低 |
| 股东强 | 主力机构/游资席位介入 |
| 成交活跃 | 换手率高，筹码流动性好 |

外因：题材大小 + 大环境（牛/熊/震荡）

---

### 3.2 量化筛选条件（供脚本实现）

**首板龙头候选（情绪复苏期）**：
```
1. 今日首次涨停（近20个交易日内无涨停记录）
2. 涨停时间早（理想：14:00 前封板）
3. 成交量 > 过去20日均量的 3 倍（爆量）
4. 市值 < 100亿（小盘容易有弹性）
5. 所在板块今日涨停个股 ≥ 3 只（板块效应）
6. 排除ST股/退市预警股
```

**连板龙头候选（上升期）**：
```
1. 近5日连续涨停（连板数 ≥ 2）
2. 高度股（全市场连板高度前3名）
3. 封板资金大（封板时成交量/成交金额大）
4. 属于主线板块（板块内涨停家数多）
5. 竞价（次日开盘前）仍在涨停价附近高开
```

**首阴买点（主升期后调整买点）**：
```
1. 曾经是连板龙头（连板 ≥ 3 板）
2. 出现首次阴线（第一个非涨停日）
3. 当日最低不破前一个涨停板的收盘价
4. 成交量较涨停日缩量（无大资金出逃）
5. 板块情绪仍在上升期
```

## 四、买卖点体系

### 4.1 买点（五大经典买点）

| 买点类型 | 触发条件 | 适用市场 |
|---------|---------|---------|
| **情绪冰点低吸** | 上涨家数 < 1000，逆势抗跌强势股 | 所有市场 |
| **板块分歧低吸** | 爆发次日，前排龙头回调不破前低 | 强势市场 |
| **弱为强之胆** | 后排股走强时，低吸还没大涨的龙头 | 强势市场 |
| **回流日低吸** | 爆发日后3日内，领涨股首次回调 | 强势市场 |
| **首阴低吸** | 连板龙头首次阴线，缩量不破前板收盘价 | 上升/补涨期 |

---

### 4.2 卖点（四大见顶信号）

| 信号 | 具体表现 |
|------|---------|
| **高位缩量加速** | 领涨股高位一字板涨停（成交量反而很小）→ 做多情绪极致 |
| **放量剧震** | 高位放量滞涨/放量大阴线/开板闷杀 |
| **大盘股补涨** | 板块内大市值股票开始补涨（资金末端接力信号）|
| **亏钱效应扩散** | 板块内中位股持续掉队，亏钱效应从外围向龙头蔓延 |

---

### 4.3 板块切换信号

**新题材接力需满足（3条）**：
1. 板块内有大票放量启动
2. 板块内涨停个股多且持续维持热度
3. 板块内出现市场有辨识度的高位股，与老题材龙头走出相反交叉走势

**老题材退潮信号（3条）**：
1. 高位股出现一字板/放量剧震等见顶信号
2. 跟风股掉队，亏钱效应扩散
3. 板块指数滞涨、成交额萎缩

---

### 4.4 补涨龙特征

当总龙头筑顶 → 踏空资金寻找补涨龙：
- **特征**：启动后不像龙头那样试探性拓展高度，而是**对标龙头一口气冲顶**
- **规律**：总龙头龙回头结束（首阴）→ 补涨龙见顶，几乎同时
- **操作**：总龙头首阴出现后，立即回避补涨龙（风险极高）

## 五、行情分类与仓位策略

### 5.1 强势市场（重仓主升）

**判断标准**：指数上升或企稳 + 主线清晰 + 龙头明确

| 要素 | 操作 |
|------|------|
| 仓位 | 重仓（70-90%） |
| 逻辑 | 做最强板块最强个股，重仓参与涨幅和成交金额前列的龙头股 |
| 注意 | 去弱留强切换要快；不持跟风股/弱势股；不参与成交小的高位连板 |
| 买点 | 情绪冰点低吸 / 板块分歧低吸 / 弱为强之胆 / 回流买点 |
| 卖点 | 板块高潮（后排大涨）/ 中位股亏钱效应扩散 / 大盘股补涨 / 跌破5日线 |

---

### 5.2 震荡市（降仓博弹性）

**判断标准**：指数稳定 + 无主线 或 多板块同时异动

| 要素 | 操作 |
|------|------|
| 仓位 | 中等（30-50%） |
| 逻辑 | 利用多热点跷跷板效应，滚动操作各板块核心人气股 |
| 注意 | 不重仓题材股；以隔日超短为主，见好就收；不早盘追高（容易杀尾盘） |
| 买点 | 情绪冰点低吸 / 个股缩量回踩 / 盘中急跌低吸 / 尾盘低吸 |
| 卖点 | 情绪高潮 / 板块高潮 / 分时急拉均线不跟 / 跌破5日线 |

---

### 5.3 弱势市场（空仓休息或轻仓试错）

**判断标准**：指数下跌空头排列 + 市场无主线 + 亏钱效应明显

| 要素 | 操作 |
|------|------|
| 仓位 | 轻仓（0-20%） |
| 逻辑 | 空仓休息为主；轻仓参与冰点后反弹；试错潜在主线 |
| 注意 | 只可低吸套利，不追求持续性；绝对低位低吸次日兑现；回避突破形态 |
| 买点 | 大票：超跌+止跌信号；冰点日；大阴线次日二次探底。小票：情绪冰点低吸 |
| 卖点 | 情绪高潮 / 个股大涨 / 跌破5日线 / 绝对止损点 |

## 六、盯盘和复盘

### 6.1 盯盘三要素

**① 看大势（大盘趋势）**
- 避开指数主跌浪（震荡和上升才适合短线）
- 顺情绪周期：上升期做多，退潮期休息
- 顺风格：哪个指数强就做哪个方向

**② 看板块联动**
- 当日领涨板块 / 领跌板块 → 找热点和亏损点
- 区间统计（近5-10日）涨幅靠前板块 → 找持续性
- 重要规律：**最强个股出自最强板块**
- 买卖参考：
  - 后排股走强 → 低吸龙头（弱为强之胆买点）
  - 跟风股掉队 → 卖出龙头（失去板块助攻信号）

**③ 看个股强弱**
- 市场异动股（当日涨停/强势异动）
- 近期涨幅最高个股是否有持续性
- 昨日连板指数（**A股最强指数**）：
  - 上升 → 连板股有赚钱效应，可打板
  - 横盘/回踩10日/20日线 → 打板无溢价，慎入

---

### 6.2 每日复盘框架

**第一步：定义行情**

每天给行情一个定义（凌驾于个股之上，是最重要的指导）：

```
当前情绪处于什么阶段？（冰点/复苏/发酵/高潮/退潮）
老周期还是新周期？
主线是什么？
龙头是谁？
```

**第二步：情绪数据统计**
- 涨停家数 / 跌停家数
- 连板家数 / 最高连板数（当日市场高度）
- 涨跌家数（通达信 880005）
- 昨日连板指数走势

**第三步：板块分析**
- 今日涨停最多的板块（找主线）
- 今日异动的新板块（找新龙）
- 区间统计近5日 / 近10日涨幅最大的板块

**第四步：个股分析（只看核心龙头）**

针对全市场涨停股，逐一记录：
```
封板时间 | 连板高度 | 市值 | 所属板块 | 成交金额
```

筛选核心：最强板块的最强个股 = 重点跟踪对象

> 对于跟风股、杂毛股：不看、不关注、不研究、不买卖

**第五步：次日计划**

根据行情定义，提前梳理：
- 情绪高潮 → 次日去弱换强
- 情绪冰点 → 次日低吸强势股
- 板块分歧 → 次日做前排龙头回流
- 前龙见顶 → 次日寻找新龙启动信号

## 七、龙头战法口诀

```
顺周期，顺趋势，顺风格；
做最强，抓领涨，跟情绪。

冰点低吸，高潮离场；
分歧去弱，回流加仓。

龙头三要素：提前启动，板块联动，极限成交量。
见顶四信号：缩量加速，剧震，大盘股补涨，亏钱效应扩散。

最强板块的最强个股，就是龙头。
不拘形式，不拘风格，跟随市场赚钱效应。
```

## 八、量化实现说明（配合 scan_dragon.py）

### 可量化的核心指标

| 指标 | 数据来源 | 量化方法 |
|------|---------|---------|
| 涨停家数（情绪） | akshare `stock_zh_a_spot_em` | 统计涨跌幅 = +10% 的数量 |
| 跌停家数 | 同上 | 统计涨跌幅 = -10% 的数量 |
| 连板数 | akshare `stock_zt_pool_em` | 获取涨停板股票池 |
| 市场高度 | 涨停板历史数据 | 统计最近5日连续涨停最高次数 |
| 成交量异常放大 | akshare 历史日线 | 今日成交量 / 20日均量 |
| 首板候选 | 涨停 + 近期无涨停记录 | 过滤连板股得首板 |
| 抗跌强势股 | 大盘跌时个股涨幅 | 大盘下跌日个股涨跌幅排名 |

### 无法量化（需人工判断）

- 题材质地（是否是市场热点）
- 分时走势强弱（盘中主力行为）
- 板块联动共振程度（需盘面感知）
- 龙头地位（是否真正引领而非跟风）
